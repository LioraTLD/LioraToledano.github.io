Le site existant présentait une navigation difficile d’utilisation pour les internautes.
Le but de ce projet a été d’élaborer une arborescence optimisée et simplifiée afin de rendre la navigation plus simple, intuitive et lisible pour les utilisateurs qui pour la plupart d’entre eux ne sont pas à l’aise avec l’informatique et les nouvelles technologies.
Pour réaliser ce projet j'ai eu le choix de modifier les pages du site à partir de l'existant ou bien de créer un nouveau site.
J'ai priviligiée comme solution le commencement d'un projet vierge.

PRÉSENTATION DU PROJET DE STAGE DE 2ÈME ANNÉE

Lors de mon stage de 2 ème année, j'ai réalisé un site web pour un hôtel.

Le dirigeant de l'entreprise d'AZGAR PROMOTION est propriétaire d'hôtels situé dans de nombreuses villes en Espagne.
Concrètement, le projet qui m'a été attribué fut de concevoir un site web permettant d'informatiser la disponibilité des chambres en fonction des périodes saisies par les utilisateurs.
Jusqu’à présent, la gestion des disponibilités des chambres et les réservations en ligne se faisaient par le biais des plateformes Onlime Tourism Agencies telles que Booking, TripAdvisor, Abritel, PAP.
Cependant, le monde de l'informatique étant en perpétuelle évolution, il était alors important de créer sa propre plateforme en ligne en fonction des besoins précis de l'entreprise afin de ne plus dépendre des plateformes externes, compliquées d'utilisation et souvent mal pensées.

ENVIRONNEMENT TECHNIQUE

Pour mener à bien la réalisation de ce projet j'ai utilisé:



Framework SYMFONY

SYMFONY est un est un ensemble de composants écrit utilisant le langage en PHP.

Une framework a pour but de fournir des fonctionnalités modulables et adaptables permettant de faciliter et d'accélérer le développement d'un site web


Installation d'un plugin pour le design du site

Afin que le design du site soit attirant, j'ai installé un plugin.

Pour rappel, un plugin est l'installation d'une extension permettant de greffé un logiciel à un autre logiciel à travers une interface prévue à cet effet, et apporter à ce dernier de nouvelles fonctionnalités.


PhpStorm

J'ai utilisé l'éditeur de code PhpStorm .

PhpStorm est un éditeur de texte pour le langage PHP et HTML, CSS, JavaScript qui sont des langages orientés interface.
